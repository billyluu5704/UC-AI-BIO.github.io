abstract: Lack of robustness is a common issue exhibited by many machine learning models. Consequently, a means of assessing model robustness is a useful design tool. This work uses rigorous verification techniques to assess the robustness of fuzzy trees trained on the Breast Cancer Wisconsin (Original) dataset. Breast cancer classification as malignant or benign clearly benefits from a robust classifier. A constrained implementation of Adaptive Neuro-Fuzzy Inference Systems (ANFIS) is used to facilitate both verification and gradient-based training of each tree. The resulting fuzzy trees are checked for classification robustness at each data point when the input is subjected to bounded perturbations. Robustness is assessed by each of two verification approaches, CROWN and α-CROWN. These provide bounds on model outputs with varying levels of conservativeness; α-CROWN iteratively improves the bounds produced by CROWN. It is shown that higher nominal performance does not equate to greater robustness. Tradeoffs are examined between model robustness, model size, and compute time required for verification.
